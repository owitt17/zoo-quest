## ZooQuest

ZooQuest is a browser-based quiz adventure game built with plain HTML, CSS, and JavaScript. Players follow story cutscenes, navigate menus, and answer timed questions with a lives system. The project uses local images, audio, and video assets to create a full arcade-style experience.
